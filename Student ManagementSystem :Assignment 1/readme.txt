Project Description

This is a simple Java CRUD (Create, Read, Update, Delete) project made using Core Java.

The project allows users to:

Add Student
View Students
Update Student
Delete Student

The project uses:
Classes and Objects
ArrayList
Scanner
Methods
Loops
CRUD Operations
Project Structure
StudentManagementSystem/

├── src/
│   ├── Student.java

└── README.txt
File Explanation:
Student.java

This file stores student data such as:

ID
Name
Course

It acts as a blueprint/template for student objects.
StudentManager.java

This file performs CRUD operations:

Add Student
View Student
Update Student
Delete Student

It stores student objects using ArrayList.
Main.java

This is the main file of the project.

It:

Takes user input
Creates student objects
Calls CRUD methods
Displays output
